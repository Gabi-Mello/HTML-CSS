tipos de "rel" e por que

  • rel='next', indica que é um link para outra parte do documento atual
  • rel='prev' indica que o link vai para a parte anterior
  • rel=author indica que é link para o autor do artigo atual
  • rel=external indica que é para outro site
  • rel="nofollow" indicar que é para um site não endossado, como um link pago ou um site que não validamos

tipos de Target=

  • _blank, opens up on a new tab Google
  • _ self or no target at all, opens up on the same tab Google
  • _top breaks outside of all the frames it is nested in and opens the link as top document in the browser window Google
  • _parent open the in the next level up of a frame if they were nested to inside one another Google
